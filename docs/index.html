OmniXLF

Simple, free and cross-platform translation editor.

Supports Windows, MacOS and Linux.

Supported Translation Formats

OmniXLF currently supports XLIFF 1.2 which is the default translation file format of Angular 2.

Download

OmniXLF is freely available for Windows, MacOS and Linux.
